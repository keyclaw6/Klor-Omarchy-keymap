Features & Behavior

  • Home Row Mods: GACS order (GUI, ALT, CTL, SFT). Chordal hold — mod activates only on cross-hand chords.
  • Danish chars: Hold P/;/' for å/æ/ø. Also on RAISE layer via Unicode Map.
  • DK_SC_AE chord: Hold ; + any left-hand key = RGUI modifier (for Hyprland shortcuts).
  • One-shot shift: Tap OSM thumb → next key is shifted. Double-tap → caps lock. 3s timeout.
  • Autocorrect: 4,200+ entries, active by default. Toggle: ADJUST layer → AC key.
  • Encoders: Rotation = Volume ± on all layers. Button = MUTE (left), PLAY/PAUSE (right). Vial-remappable.
  • AF keys: LALT+F13–F23 on RAISE layer — bind to Hyprland/Omarchy actions.

Quick Reference

  • Enter command mode: Double-tap Right Alt (250ms window)
  • Stop STT recording: Press RALT or T again
  • All results: Copied to clipboard → paste with Ctrl+V
  • Caps lock: Double-tap OSM shift thumb
  • Bootloader (combo): 5× all 4 thumbs on one half (<3s)
  • Bootloader (key): LOWER+RAISE → bottom-left (BOOT)
  • Vial unlock: Hold Q + P simultaneously
  • Build: make -C ~/vial-qmk geigeigeist/klor/2040:vial
